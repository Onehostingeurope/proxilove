# PROXILOVE app Assets

Place your app icon here as `icon.png`.

Requirements:
- Size: 1024x1024 pixels
- Format: PNG with transparency support
- Background: #0A0F24 (Deep Navy)
- Design: "S" letter in #00F0FF (Electric Blue) with subtle glow effect

For Android adaptive icon:
- Place foreground layer as `adaptive-icon.png` (1024x1024)
- Background color is set in app.json: "#0A0F24"

For Splash Screen:
- The splash background color is set in app.json: "#0A0F24"
- No additional image needed for basic splash

Quick placeholder generation:
  You can generate a placeholder icon using any image editor.
  The app will still run without a real icon in development mode.
